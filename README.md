# Ön Muhasebe Öğrenme ve Hazırlık Aracı

Logo GO3 veya benzeri muhasebe programlarına geçmeden önce ön muhasebe kavramlarını öğrenmek için hazırlanmış kişisel kullanım aracı.

## Özellikler

- **4 Seviye Öğrenme:** Borç-Alacak, Cari Hesap, Fatura, KDV, Stok, Vade/Tahsilat, Kasa/Banka, Raporlar, İş Disiplini
- **İnteraktif Simülatörler:** Cari hesap ve stok pratiği
- **Quiz:** Bilgi testi
- **İlerleme Takibi:** localStorage ile kayıt

## Kullanım

Tarayıcıda `index.html` dosyasını açın veya GitHub Pages üzerinden erişin.

## GitHub Pages'e Alma

1. [GitHub](https://github.com/new) üzerinde yeni bir repo oluştur (`on-muhasebe-ogrenme` veya istediğin isim). README ekleme.
2. Proje klasöründe:
   ```bash
   git remote add origin https://github.com/KULLANICI_ADIN/on-muhasebe-ogrenme.git
   git push -u origin main
   ```
3. Repo **Settings** > **Pages** > **Source**: "Deploy from a branch" > Branch: **main** > Folder: **/ (root)** > Save
4. Birkaç dakika sonra site `https://KULLANICI_ADIN.github.io/on-muhasebe-ogrenme/` adresinde yayında olacak.
